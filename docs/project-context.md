# Jon Labs - Contexte Projet

## Vue d'ensemble

### Objectif

Créer un site web personnel/freelance pour **Jon Labs** (marque de Jonathan Vouilloz) afin d'établir une présence en ligne forte, attirer des clients, et showcaser une approche créative polyvalente.

### Philosophie

| Principe | Description |
|----------|-------------|
| Action > Perfection | Site imparfait en ligne > site parfait dans la tête |
| Contenu first | Préparer le contenu AVANT de coder |
| Authentique | Assumer la polyvalence, ne pas se limiter |
| Fancy mais pro | Casser les codes tout en restant crédible |
